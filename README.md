# 📊 Smart Dataset Analyzer V2

A modern full-stack data analysis platform built with React, FastAPI, Pandas, and Machine Learning tools.

Users can upload datasets, explore dataset statistics, visualize data, detect missing values, and generate AI-powered insights.

---

## 🚀 Current Features

### 📁 Dataset Upload

* Upload CSV files
* Dataset validation
* Secure file handling
* FastAPI backend integration
* Success and error handling
* Dataset upload workflow

## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/YOUR_USERNAME/smart-dataset-analyzer-v2.git
```

### Frontend Setup

```bash
cd frontend
npm install
npm run dev
```

### Backend Setup

```bash
cd backend
pip install -r requirements.txt
uvicorn main:app --reload
```
